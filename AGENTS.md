# AGENTS.md

## 项目目标

TrendPilot 是用于 AI 产品经理面试展示的 MVP：帮助潮流服饰品牌运营人员从经营数据中发现问题，并在后续阶段获得有依据的 AI 解释和行动建议。

## 技术与结构

- Python 3.11+、Streamlit、Pandas、pytest。
- `app.py` 只负责页面与交互，业务逻辑放在 `src/`。
- 示例数据位于 `data/`，测试位于 `tests/`，项目文档位于 `docs/`。

## 开发约束

- 严格按阶段推进，不随意扩大 MVP 范围。
- 所有确定性数字必须由 Python 计算，大模型不得负责关键数字计算。
- 优先使用透明、可解释的规则和清晰的数据证据。
- 新功能必须有测试；修改后运行全部测试。
- 不提交 `.env`、API Key、Streamlit secrets 或其他敏感信息。
- 页面与业务逻辑分离，函数职责单一，关键函数使用类型标注。
- 对上传文件等外部输入进行校验，并向用户展示友好错误，不直接展示异常堆栈。
- 所有 AI 建议必须保留人工确认机制；不得虚构用户评价、市场数据或商业效果。

## 当前阶段边界

阶段 1 只包含 CSV 加载、必填字段校验、Session State 保存、数据摘要和预览。不要提前实现指标分析、异常检测、AI 诊断、单品分析或运营方案。
